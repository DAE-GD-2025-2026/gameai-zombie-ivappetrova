# Zombie AI exam project
## Made by Iva Petrova, 2DAE11

This is my Game AI exam project. I made an AI-controlled character that tries to survive as long as possible against zombies.
The brain of the agent is built inside StudentPerceptor. It decides what the agent should do at any moment- whether that's moving around, grabbing items, or dealing with nearby zombies.
Check the PowerPoint for more details on how everything works.
